                                                *auto-nohlsearch*
INTRODUCTION                                    *vim-auto-nohlsearch*

A plugin that automatically turns off `hlsearch` when you're finished
searching a buffer.


OPTIONS                                         *vim-auto-nohlsearch-options*

*g:auto_nohlsearch_enabled*    If true, enables automatically turning off
(default = true)             hlsearch when a search ends. If false, the plugin
                             is disabled.
